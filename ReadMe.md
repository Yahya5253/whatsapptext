# WhatsApp Formatter

A minimal web app for formatting WhatsApp text with bold, italic, strikethrough, and code styles.  
No data is stored or transmitted — everything runs locally in your browser.

## Features

- Format text for WhatsApp with bold, italic, strike, and code
- Preview real WhatsApp text formatting before sending
- Copy text for WhatsApp use

## Usage

Just open the site (GitHub Pages link below), type or paste your text, select text and use formatting buttons or keyboard shortcuts, then copy for WhatsApp!
